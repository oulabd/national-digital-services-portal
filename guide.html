💡 نصائح وحيل

✓ جرب جميع التبويبات

في بوابة الجامعات والمدارس توجد تبويبات متعددة - استكشفها جميعاً

✓ ملأ النماذج

النماذج توفر تحقق حقيقي - جرب إرسال نموذج فارغ لترى الأخطاء

✓ الجداول التفاعلية

الجداول تحتوي على أزرار تفاعلية - جرب الضغط عليها

✓ النوافذ المنبثقة

اضغط Escape للخروج من أي نافذة منبثقة بسرعة

🎨 التخصيص والتطوير

تغيير الألوان

عدّل المتغيرات في css/style.css:

:root {
  --primary-olive: #2F4F3E;
  --gold: #C8A24A;
  /* عدّل أي لون هنا */
}

إضافة صفحة جديدة

  1. انسخ أي ملف HTML موجود
  2. عدّل المحتوى
  3. أضف الرابط في القائمة الجانبية

ربط قاعدة بيانات

عدّل دوال JavaScript في js/main.js للاتصال بـ API الخاص بك

🔧 حل المشاكل

❓ الصفحة لا تظهر بشكل صحيح

✓ تأكد أن جميع الملفات في نفس المجلد
✓ أعد تحميل الصفحة (Ctrl+Shift+R)
✓ افسح ذاكرة التخزين المؤقتة

❓ النماذج لا تعمل

✓ افتح console (F12) وابحث عن الأخطاء
✓ تأكد أن JavaScript مفعل
✓ جرب متصفح آخر

❓ الأزرار أو الروابط لا تستجيب

✓ تأكد من أن ملف main.js محمل بشكل صحيح
✓ افتح console وابحث عن ReferenceError
✓ تأكد من أن معرف العنصر صحيح (ID)

🎯 الخطوات التالية

  1. استكشف جميع الصفحات - تعرف على جميع الخدمات
  2. ادرس الكود - افهم كيفية عمل HTML و CSS و JavaScript
  3. عدّل وجرب - غيّر الألوان والنصوص والتخطيط
  4. أضف ميزات جديدة - أضف نماذج وجداول جديدة
  5. ربط قاعدة بيانات - اجعلها تطبيق حقيقي
  6. نشر على الإنترنت - شارك البوابة مع الآخرين
